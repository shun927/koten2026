2. コンセプト

左右の穴に人差し指を差し入れ、Boxの中で向かい合わせにする（☛ ☚）。その先のスクリーンには、この部屋の真ん中に置かれたBoxと、左右のフレーム外から侵入してくる巨大な指が映し出されている。Boxの中で触れている主観的な手と、スクリーンに映る客観的な手の間にズレが生じ、「この手は誰のものか」という問いが生まれる。
（検討中）さらに、左右から近づく音が聞こえたとき、体験者は「覗く側」と「覗かれる側」の境界に立たされる。


3. 構成

```

[体験者]

    ↓ 左右の手（今回はひとさし指をさした状態（☛ ☚）で）を入れる

[Box（台座の上）] 

    ├─ 内部：明るく照明（カメラ認識用）

    ├─ 外装：遮光性（外からは中が見えない）

    └─ 正面：開口部（覗き穴または開放？）

           ↓

    [カメラ（正面）]

           ↓

[スクリーン（壁面）]

    └─ 「部屋の真ん中にBoxがある」俯瞰視点または正面視点の映像

```

4. 映像・システム構成

映像内容：
俯瞰視点で「部屋の中のBox」を映す。現実の手の動きに連動し、左右のフレーム外から巨大な指（CGゆび×2）がBoxに向かって伸びてくる。（わずかな手ブレやレンズ感を加え、客観的な監視カメラのような視点を演出。）

手の表現：
現実の左右の手をトラッキングし、スクリーン上では左右のフレーム外から巨大な指（現実の手と同じ動き）が侵入してくる

視点：
「第三者が見ている」ようなわずかな手ブレやレンズ感を加え、客観視の感覚を強調？。

BOX: 
無機質で周りからライトを当てて、神聖な物っぽくする。

サウンド : 
・「箱の内部」の音響：
Boxに手を入れた瞬間だけ、反響（リバーブ）が極端に変化する（閉鎖空間に入った感）
・「部屋の音」の消滅：
手を入れている間、外部の環境音が遮断され、「自分の心臓の音」や「血流の音」だけが聞こえる（内面への没入）


5. 体験の流れ（時間軸）

1. 【侵入】 左右から手をBoxに入れる。中は明るく、両手を向かい合わせ（☛ ☚）にする。

2. 【発見】 正面のスクリーンを見ると、そこには「この部屋の真ん中に置かれたBox」と、そこに差し込まれた巨大な指が映っている。

3. 【ズレ】 指を動かすと、スクリーンの指も動く。しかしそのスケールや視点は、まるで自分が巨人になったかのように、あるいは誰かの眼を通して見ているように感じられる。

4. 【逆転】 左右から近づく音が聞こえる。自分がBoxの中を覗く能動的な側であると同時に、部屋という大きなBoxの中に閉じ込められて外から覗かれている受動的な側でもあることに気づく。（検討）





6. テーマ

入れ子構造（ドロステ効果）：
Boxは部屋の模型であり、部屋はBoxの拡張である。体験者はその境界に位置する。

先入観の崩壊：
「自分の身体は自分のものである」という無意識の先入観が、予測と感覚のズレによって一時的に解ける。

自己の再帰：
スクリーンに映る「自分」を見ることで、主体（見る側）と客体（見られる側）の入れ子が生じ、自己というシステムの不安定さを発見する。

（検討中）:
誰かが部屋の外からこちらを見ていたら？体験者自身が「箱の中身はなんだろな」される側になる逆転。
左右から手をboxに入れる（能動）
→スクリーンを見ると、そこには「この部屋を外から見た俯瞰図」があり、自分が手を入れているboxが部屋の真ん中にある（受動）
→同時に、部屋の左右から「何かが近づく音」が聞こえる（自分が観察されている緊張感）
